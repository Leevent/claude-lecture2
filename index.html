FAQ

常見問題

Skill 和 Projects 有什麼不同？ ▾
Projects 是你的「AI 工作空間」，讓 Claude 記住你的背景和偏好——像一個跟你共事的同事。Skill 是你設計的「AI 工作 SOP」，一個指令就能觸發完整的工作流程——像一份自動化流程表。兩者搭配使用，效果最好。
MCP 串接需要寫程式嗎？ ▾
不需要！Google Calendar 和 Notion 是官方整合，在 Claude 桌面版介面一鍵授權即可。Google Sheets 需透過第三方工具，但設定過程也只是複製貼上一份純文字設定檔，不需要寫任何程式碼。
MCP 和 n8n / Zapier 有什麼不同？ ▾
MCP 是 Claude 的「手和眼」，即時互動——你下指令，Claude 當場幫你操作。n8n / Zapier 是排程型自動化，設定好後在背景自動觸發。不是替代關係，是互補：MCP 做臨時決策，n8n 做重複排程。
使用 MCP 需要付費版 Claude 嗎？ ▾
MCP 串接功能需要 Claude Pro 以上方案及 Claude 桌面版 App。Skill 設計本身在免費版也能使用，只是進階功能（Extended Thinking、更多對話量）需要 Pro 版。
MCP 串接公司資料安全嗎？ ▾
MCP 的權限由你控制——Google Calendar 和 Notion 都是 OAuth 授權，你可以選擇分享哪些日曆或頁面。Claude 不會永久儲存你的資料。建議先從個人資料開始練習，確認流程後再決定是否串接工作相關內容。
5 級 Skill 成熟度要多久才能從 L1 到 L5？ ▾
L1→L2 通常幾天內就能達成，關鍵是加入變數讓 Skill 更靈活。L2→L3 需要學會 MCP 串接（本站有完整指南）。L3→L4 需要設計多步驟工作流，通常需要 1-2 週實踐。L5 涉及讓多個 AI 助理分工合作，屬於進階內容。
學完 B 班之後呢？ ▾
B 班學完 Skill + MCP + Cowork 三大核心後，你已經能設計完整的 AI 工作系統。想更進一步，可以關注講師的部落格和社群，未來會有更多進階內容（例如讓 AI 直接幫你寫程式、建系統、跑自動化）。

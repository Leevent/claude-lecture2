Level B — 手動設定

🔵 編輯 config.json：解鎖所有 MCP Server

跟著以下步驟操作，不需要寫程式

1

找到設定檔位置

Claude Desktop 的 MCP 設定存在這個 JSON 檔案裡：

作業系統檔案路徑
macOS~/Library/Application Support/Claude/claude_desktop_config.json
Windows%APPDATA%\Claude\claude_desktop_config.json
💡 如果檔案不存在，自己新建一個空的就好。
2

範例：加入 Filesystem Server

最基本的 MCP Server——讓 Claude 讀寫你指定的資料夾：

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/你的名字/Documents"]
    }
  }
}
⚠️ 路徑請改成你自己的資料夾。只授權需要的目錄，不要給根目錄。
💡 npx = Node.js 套件執行器（適用大多數 MCP Server）。uvx = Python 套件執行器（少數 Server 使用）。docker = 容器化執行（最隔離但需安裝 Docker）。
不確定用哪個？照 Server 文件的範例複製即可。
3

範例：同時加多個 Server

一個 config.json 可以同時串接多個 MCP Server：

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/你/Documents"]
    },
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "brave-search": {
      "command": "npx",
      "args": ["-y", "@anthropic-ai/mcp-server-brave-search"],
      "env": { "BRAVE_API_KEY": "你的 API Key" }
    }
  }
}
4

儲存 → 重啟 → 測試

儲存 config.json 後，完全關閉 Claude Desktop 再重新開啟。在對話框輸入：

「請列出我 Documents 資料夾裡的所有檔案」
「搜尋最新的 MCP 相關新聞」
「記住：我偏好繁體中文回覆」
💡 如果 Claude 說找不到工具，檢查 JSON 格式（逗號、大括號）是否正確。推薦用 jsonlint.com 驗證。

Security Awareness

🛡️ 安全提醒

⚠️ 43% 被測試的第三方 MCP Server 存在安全漏洞 （來源：Invariant Labs, 2026）

安裝前請注意：

  • 優先使用 Anthropic 官方合作夥伴（一鍵授權那 11 個最安全）
  • 第三方 Server 檢查 GitHub 星數、最近更新、Issue 狀況
  • Filesystem 只開放特定資料夾，不要授權根目錄
  • 需要 API Key 的 Server，Key 不要外洩或 commit 到 Git
  • 社群建議：從 1-2 個 Server 開始，逐步擴展

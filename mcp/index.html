Concept

MCP 和自動化工具有什麼不同？

不是替代關係，是互補：MCP 做即時決策，n8n 做重複排程

🔗 MCP（Model Context Protocol）

Claude 的「手和眼」——你下指令，Claude 當場幫你操作。像是有個助理坐在你旁邊，你說什麼他馬上做。

即時互動

⚙️ n8n / Zapier

排程型自動化——設定好後在背景自動觸發。像是預設好的流水線，固定時間固定動作。

🔗 入門三件套：第一步從這裡開始

Claude 桌面版官方整合，點擊卡片展開串接步驟

📅

Google Calendar

讓 Claude 看你的行程、安排時間、提醒你重要會議

點擊查看串接步驟 →
1
開啟 Claude 桌面版 Settings → Integrations
2
找到 Google Calendar，點擊 Connect（官方整合，一鍵授權）
3
Google 帳號授權（選擇要分享的日曆）
4
測試：問 Claude「我明天有什麼行程？」
5
進階：讓 Claude 幫你找空檔、建立新事件
🔒 權限為授權範圍內的唯讀/讀寫，Claude 不儲存你的日曆資料
📝

Notion

讓 Claude 讀寫你的知識庫，直接在 Notion 建立頁面和資料庫

點擊查看串接步驟 →
1
開啟 Claude 桌面版 Settings → Integrations
2
找到 Notion，點擊 Connect（官方整合，一鍵授權）
3
Notion 授權（選擇要分享的工作區和頁面）
4
測試：問 Claude「幫我在 Notion 建一個會議紀錄」
5
進階：搭配 Skill 自動整理筆記到指定資料庫
🔒 僅存取你授權的頁面，不會碰其他工作區內容
📊

Google Sheets

讓 Claude 直接操作你的試算表，分析資料、生成報表

點擊查看串接步驟 →
1
此工具需透過第三方 MCP Server 串接（非官方一鍵整合）
2
需設定 config.json（不需要寫程式，照步驟即可完成）
3
Google 帳號授權（選擇要分享的試算表）
4
測試：問 Claude「幫我分析這張表的銷售趨勢」
5
進階：自動產生月報表、資料視覺化
⚙️ 與 Calendar/Notion 不同，Sheets 需要額外設定 MCP Server

Security Awareness

🛡️ 安全提醒

⚠️ 43% 被測試的 MCP Server 存在 Command Injection 漏洞

MCP 生態系仍在快速發展中，安裝第三方 Server 前請注意：

  • 優先使用 Anthropic 官方合作夥伴（上方 11 個），安全性有保障
  • 第三方 Server 請檢查 GitHub 星數、最近更新時間、Issue 狀況
  • 避免授予不必要的權限（如 Filesystem 只開放特定資料夾）
  • 企業環境請經過 IT/Security 審核再部署
  • 社群建議：從 1-2 個 Server 開始，逐步擴展

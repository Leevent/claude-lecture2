How to Use

使用方法

1

盤點 Projects

列出你日常使用 Claude 的工作領域。例如：社群經營、客戶提案、會議記錄等。

2

設計 Skills

針對每個 Project 的高頻重複任務，設計對應的 Skill。用 Skill 工坊完成設計。

3

串接 MCP

思考哪些 Skill 需要連接外部工具才能真正自動化。參考 MCP 指南完成串接。
